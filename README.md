### Configuration

1. Create a `.env` file in the root directory of the project.
2. Add necessary configuration variables in the `.env` file.
   Example:

   ```bash
    PORT=5000
    DB_URL=your_db_connection_uri
    STORE_ID="###"
     <!-- Payment https://www.aamarpay.com/ -->
    SIGNATURE_KEY="#########"
    PAYMENT_URL="####"

   ```
